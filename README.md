# FlowState

<h2>Our solution to getting the most out of your day.</h2>

<p>FlowState is a web application that helps you stay organized and focused on your tasks. It uses a simple and intuitive "drag-and-drop" interface to help you manage your daily tasks and stay on top of your goals.</p>

<p>FlowState is built with React, FastAPI, and MongoDB. It is a Web App (PWA) that you can access with any type of account.</p>

<p>FlowState is a product of the 9th Annual Hacklahoma Hackathon and is still being improved upon for deployment.</p>

<p>To operate, you will need to install .venv, activate it, pip install -r requirements.txt, and then run main.py. To activate frontend, cd frontend, npm install, and then npm run dev.</p>
